## 📊 Projeto de Análise de Dados em Python

Este projeto foi desenvolvido com o objetivo de praticar e consolidar habilidades em análise de dados utilizando Python.

Durante o desenvolvimento, foram aplicadas técnicas de manipulação, tratamento e visualização de dados, utilizando as principais bibliotecas do ecossistema:

* **Pandas**: para limpeza, transformação e análise dos dados
* **Matplotlib** e **Seaborn**: para criação de visualizações e exploração gráfica

O projeto inclui:

* Tratamento e organização dos dados
* Análise exploratória (EDA)
* Geração de gráficos para identificação de padrões e insights

🎯 **Objetivo**: evoluir minhas habilidades práticas em análise de dados e me preparar para oportunidades como Analista de Dados Júnior.

📌 Este projeto faz parte do meu processo contínuo de aprendizado, com foco em aplicar conceitos teóricos em cenários práticos.
